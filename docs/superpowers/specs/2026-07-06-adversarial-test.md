1. **PPTTemplateSelector** — 组件完整，0引用

3. **5种高级数据源** — 后端完整，前端无UI
4. **HTML脑图导出** — 后端完整，ExportPanel未列出
5. **大纲内联编辑** — 设计中有，未实现
6. **撤销UI** — hook存在，无按钮触发
